# Knowledge Bases

Curated knowledge bases for Open WebUI RAG integration.

## Contents

| Directory | Topic | Files | Description |
|-----------|-------|-------|-------------|
| `chemistry/` | Chemistry | 18 | Atomic structure through quantum chemistry, undergraduate/graduate level |
| `bankruptcy/` | California Bankruptcy Law | 15 | Chapter 7/13, exemptions, means test, adversary proceedings, post-bankruptcy |
| `kali/` | Kali Linux | 20 | Installation, tools, networking, exploitation, forensics, wireless |

## Usage

Each directory is designed to be uploaded as a separate knowledge base in Open WebUI:

1. Admin -> Knowledge -> Create Knowledge Base.
2. Upload all Markdown files from one directory.
3. Attach the knowledge base to a workspace model.

## Open WebUI Deployment

These KBs are intended for deployment at `ziaventuregroup.cloud`:

| Knowledge Base | Workspace Model |
|----------------|-----------------|
| Chemistry Reference | The Chemist |
| California Bankruptcy Law | Legal Advisor |
| Kali Documentation | Brutus Maximus |

## Security

Do not commit credentials, API tokens, passwords, private case documents, or operational secrets to this public repository.
