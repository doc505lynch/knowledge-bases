# Knowledge Bases

Curated knowledge bases for Open WebUI RAG integration.

| Directory | Topic | Files | Description |
|-----------|-------|-------|-------------|
|  | Chemistry | 18 | Atomic structure through quantum chemistry — undergraduate/graduate level |
|  | California Bankruptcy Law | 15 | Chapter 7/13, exemptions, means test, adversary proceedings, post-bankruptcy |
|  | Kali Linux | 20 | Installation, tools, networking, exploitation, forensics, wireless |

## Usage

Each directory is designed to be uploaded as a separate knowledge base in Open WebUI:

1. Admin → Knowledge → Create Knowledge Base
2. Upload all  files from a directory
3. Attach the knowledge base to a workspace model
