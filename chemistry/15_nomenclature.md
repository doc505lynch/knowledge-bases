# Chemical Nomenclature

## IUPAC Nomenclature System

The International Union of Pure and Applied Chemistry (IUPAC) provides systematic naming rules for all chemical compounds.

## Inorganic Nomenclature

### Binary Ionic Compounds (Metal + Nonmetal)
**Format**: [cation name] [anion name with -ide suffix]

- NaCl → sodium chloride
- MgBr₂ → magnesium bromide
- Al₂O₃ → aluminum oxide
- CaS → calcium sulfide

**Variable-charge metals**: include oxidation state in Roman numerals
- FeCl₂ → iron(II) chloride (ferrous chloride, old system)
- FeCl₃ → iron(III) chloride (ferric chloride, old system)
- CuO → copper(II) oxide (cupric oxide)
- Cu₂O → copper(I) oxide (cuprous oxide)

**Pattern for oxyanions**:  
- Base: -ate (most common oxidation state of nonmetal)
- One fewer O: -ite
- One more O than -ate: per___ate  
- One fewer O than -ite: hypo___ite

**Hypo**chlor**ite** (1 O) → chlor**ite** (2 O) → chlor**ate** (3 O) → **per**chlor**ate** (4 O)

### Acids

**Binary acids** (HX dissolved in water):  
hydro[nonmetal name without -ine]ic acid  
- HCl(aq) → hydrochloric acid
- HBr(aq) → hydrobromic acid
- HF(aq) → hydrofluoric acid
- H₂S(aq) → hydrosulfuric acid

**Oxyacids** (H + polyatomic ion with O):  
- -ate ion → -ic acid: H₂SO₄ → sulfuric acid; HNO₃ → nitric acid; H₃PO₄ → phosphoric acid
- -ite ion → -ous acid: H₂SO₃ → sulfurous acid; HNO₂ → nitrous acid
- per___ate → per___ic acid: HClO₄ → perchloric acid
- hypo___ite → hypo___ous acid: HClO → hypochlorous acid

### Bases
NaOH → sodium hydroxide  
Ca(OH)₂ → calcium hydroxide (lime, slaked lime)  
NH₃(aq) → ammonia solution (ammonium hydroxide, NH₄OH — not a true hydroxide)  
Al(OH)₃ → aluminum hydroxide

### Hydrates
Ionic compounds with water of crystallization; number by Greek prefix:
- CuSO₄·5H₂O → copper(II) sulfate pentahydrate (blue vitriol)
- Na₂CO₃·10H₂O → sodium carbonate decahydrate (washing soda)
- MgSO₄·7H₂O → magnesium sulfate heptahydrate (Epsom salt)
- CaCl₂·2H₂O → calcium chloride dihydrate
- BaCl₂·2H₂O → barium chloride dihydrate

### IUPAC Steps for Organic Compounds

1. **Identify principal characteristic group** (highest priority gets suffix):
   Priority: acids > esters > amides > aldehydes > ketones > alcohols > amines

2. **Find longest chain** containing the principal group (parent chain)

3. **Number the chain** to give lowest locants to: principal group → double/triple bonds → substituents

4. **Name substituents** alphabetically with locant numbers

5. **Complete name**: substituents (alphabetically) + parent + suffix

### Stereodescriptors

**R/S for chiral centers**:  
1. Assign priorities by CIP rules (higher atomic number = higher priority; ties: go to next atom)  
2. Orient so lowest priority (4) points away  
3. R (rectus): 1→2→3 clockwise; S (sinister): counterclockwise

**E/Z for double bonds**:  
E (entgegen): high priority groups on opposite sides  
Z (zusammen): high priority groups on same side

**cis/trans**: for ring substituents or when priorities same on each carbon  
cis: same side; trans: opposite sides

**α/β**: for sugars/steroids (β above plane, α below in ring representation)

## Coordination Compound Nomenclature

### Rules
1. Cation named before anion
2. Within complex: ligands alphabetically, then metal with oxidation state
3. Ligand prefixes: di, tri, tetra for simple; bis, tris, tetrakis for complex
4. Metal in anionic complex: Latin name + -ate (e.g., ferrate, cuprate, plumbate)
5. Metal in cationic/neutral complex: English name (iron, copper, lead)

### Nucleotide/DNA/RNA Naming
- Nucleoside = sugar + base (no phosphate): adenosine, guanosine, cytidine, thymidine, uridine
- Nucleotide = nucleoside + phosphate(s): AMP, ADP, ATP; GMP, GDP, GTP; etc.
- Deoxy prefix (d): dATP, dGTP, dCTP, dTTP (in DNA synthesis)
- RNA: AMP, GMP, CMP, UMP

### Enzyme Nomenclature (EC System)
Enzymes named by substrate + reaction type + "-ase"
EC classification by reaction type:
1. **Oxidoreductases**: oxidation/reduction (EC 1)
2. **Transferases**: transfer groups (EC 2)
3. **Hydrolases**: hydrolysis (EC 3)
4. **Lyases**: add/remove groups without hydrolysis (EC 4)
5. **Isomerases**: structural rearrangement (EC 5)
6. **Ligases**: bond formation using ATP (EC 6)
7. **Translocases**: ion/molecule movement across membranes (EC 7)

EC number example: Lactate dehydrogenase = EC 1.1.1.27
(EC class.subclass.subsubclass.serial number)
