# Kali Linux — Introduction

Source: https://www.kali.org/docs/introduction/what-is-kali-linux/

---

## What is Kali Linux?

Kali Linux is an open-source, Debian-based operating system designed for cybersecurity professionals. Originally derived from BackTrack Linux, it provides advanced penetration testing and security auditing capabilities. The platform operates across multiple architectures and remains completely free of charge.

The distribution includes several hundred specialized tools, configurations, and scripts tailored for security work, enabling professionals to focus on tasks like:

- Computer forensics
- Reverse engineering
- Vulnerability detection and exploitation
- Penetration testing
- Security auditing

## Key Characteristics

### Accessibility and Community
- Open development model with source code available via GitLab repositories
- Transparent development using multiple security protocols for all repository changes
- Completely free and open source

### Technical Foundation
- Adheres to the Filesystem Hierarchy Standard (familiar to all Linux users)
- Custom kernel with wireless injection patches for wireless assessments
- Every package is GPG-signed by individual developers

### Hardware Support
- Supports extensive device compatibility
- ARM-based systems: Raspberry Pi, BeagleBone Black, and many more
- x86, x64, and ARM architectures

### Customization
- Users can modify virtually every component, including the kernel itself
- Accommodates different professional preferences and requirements

## Target Audience

Kali Linux explicitly assumes users possess existing Linux knowledge and experience. The platform serves:

- Experienced security professionals
- Penetration testers
- Security researchers
- Informed enthusiasts

**It is not recommended for Linux beginners as a primary OS.**

## History

Kali Linux was derived from BackTrack Linux, consolidating and refining the BackTrack toolkit into a Debian-based distribution with professional-grade packaging, signing, and development practices.
